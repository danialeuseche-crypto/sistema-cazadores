# ⚡ SISTEMA DE CAZADORES LITERARIOS ⚡
### 🛡️ Gremio Alfa — Panel de Control de Manuscritos

¡Bienvenido al **Sistema de Cazadores Literarios (S.C.L)**! Esta es una plataforma web interactiva diseñada para gestionar rangos, roles de combate y un repertorio rúnico de obras literarias (Fantasía Oscura, Terror, Acción Marcial y Fanfiction).

---

## 🚀 Características del Sistema

* **Estado del Cazador:** Elige tu clase de combate anime (Luchador, Asesino, Hechicero, Invocador, Tanque) y ejecuta el despertar para descubrir tu rango aleatorio.
* **Gestión de Gremios:** Afíliate a facciones oficiales como el *Gremio Colmillo Blanco* o *Cazadores de Almas*.
* **Repertorio de Obras:** Una estantería visual interactiva que genera portadas automáticas con estilos neón según la categoría del relato.
* **Inyección de Manuscritos:** Panel dinámico para publicar nuevas historias en tiempo real y acumular EXP.
* **Aura de Concentración:** Reproductor de audio integrado para canalizar tu energía mientras redactas o lees.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica del panel y la barra lateral de navegación.
* **CSS3:** Interfaz oscura estilizada con estética Cyber-Anime, efectos neón y diseño responsivo para móviles.
* **JavaScript (Vanilla):** Lógica del sistema para el filtrado de historias, cálculo de rangos, buscador de cazadores y renderizado dinámico de la estantería.
